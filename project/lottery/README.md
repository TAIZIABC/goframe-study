# lottery — 随机抽签工具

从候选名单文件中随机抽取指定数量的名字，结果去重，并将抽签记录写入日志。

## 使用方式

```bash
# 基本用法：抽 3 人
go run main.go -file names.txt -n 3

# 带标题和自定义日志
go run main.go -file names.txt -n 2 -title "周会分享人" -log record.log
```

## 参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `-file` | (必填) | 候选名单文件（每行一个名字） |
| `-n` | `1` | 抽取人数 |
| `-title` | - | 本次抽签标题 |
| `-log` | `lottery.log` | 日志文件路径 |

## 名单文件格式

```
张三
李四
王五
# 注释行会跳过
赵六
张三    ← 重复自动去重
```

## 功能特点

- Fisher-Yates 洗牌算法，保证公平随机
- 名单自动去重 + 跳过空行/注释行
- 结果和时间**追加写入日志**，可追溯历史
- 人数超限等异常有清晰错误提示
