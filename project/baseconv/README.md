# baseconv — 进制转换工具

命令行交互式进制转换工具，支持二进制、八进制、十进制、十六进制互转。

## 使用方式

```bash
go run main.go
```

## 交互示例

```
>>> FF 16
  │  二进制 (BIN):  1111 1111
  │  八进制 (OCT):  377
  │  十进制 (DEC):  255
  │  十六进制(HEX):  FF

>>> 11111111 2
>>> 377 8
>>> -10 10       # 支持负数
>>> q            # 退出
```

## 功能特点

- 支持 **2/8/10/16** 进制输入，同时输出四种进制结果
- 交互式循环输入，输入 `q` 退出
- 支持负数
- 二进制输出每 4 位分隔，方便阅读
- 自动去除 `0x`/`0o`/`0b` 前缀
- 非法输入有清晰错误提示
