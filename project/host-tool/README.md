# host-tool — TCP 端口扫描工具

指定目标主机和端口列表，并发检测每个端口是否连通，输出状态和响应时间。

## 使用方式

```bash
go run main.go -host 127.0.0.1 -ports 22,80,443,3306,8080
go run main.go -host example.com -ports 80,443 -timeout 5s
```

## 参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `-host` | (必填) | 目标主机（IP 或域名） |
| `-ports` | (必填) | 端口列表，逗号分隔 |
| `-timeout` | `3s` | 单个端口连接超时时间 |

## 输出示例

```
扫描目标: 127.0.0.1
端口列表: 22, 80, 443, 3306, 8080
超时时间: 3s
──────────────────────────────────────────────────
  端口 22      open              耗时 636µs
  端口 80      closed            耗时 514µs
  端口 443     closed            耗时 867µs
  端口 3306    closed            耗时 873µs
  端口 8080    open              耗时 1.267ms
──────────────────────────────────────────────────
扫描完成: 5 个端口, 2 open, 3 closed
```

## 功能特点

- 所有端口**并发扫描**（goroutine + WaitGroup），速度极快
- 彩色输出（绿色 open / 红色 closed）
- 显示每个端口的连接耗时（精确到微秒）
- 结果按端口号排序
- 最后汇总 open/closed 数量
