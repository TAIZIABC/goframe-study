# weather — 天气查询工具

输入城市名，调用 wttr.in 免费 API 获取当前天气信息，终端简洁显示。

## 使用方式

```bash
# 命令行直接查询
go run main.go Beijing
go run main.go London

# 交互模式（持续输入）
go run main.go
```

## 输出示例

```

  │  🌫️  Haze

  │  💨 风速:     4 km/h (ENE)
  │  👁️  能见度:   4 km
  │  🔽 气压:     1013 hPa
  │  🔆 紫外线:   3
```

## 功能特点

- 调用 **wttr.in** 免费 API，无需注册 API Key
- 支持**中英文城市名**（Beijing / 上海 / Tokyo / London）
- 两种模式：命令行参数直接查询 + 交互式持续输入
- 显示温度、体感温度、湿度、风速风向、能见度、气压、紫外线指数
- 根据天气状况自动匹配 emoji 图标
- 10 秒超时，网络异常有清晰错误提示
