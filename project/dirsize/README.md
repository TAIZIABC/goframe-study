# dirsize — 目录大小扫描工具

扫描指定目录，列出每个子目录和文件的大小，按大小降序排列，以人类可读格式显示。

## 使用方式

```bash
go run main.go /path/to/dir          # 直接传参
go run main.go -dir /path/to/dir     # flag 方式
go run main.go                       # 默认扫描当前目录
```

## 输出示例

```
目录: /Users/xxx/goframe-study
────────────────────────────────────────────────────────
  名称                                           大小  类型
────────────────────────────────────────────────────────
  todo-web/                              56.29 MB  📁
  todo-api-js/                            5.91 MB  📁
  project/                                2.36 MB  📁
  app.js                                168.93 KB  📄
  styles.css                             25.16 KB  📄
────────────────────────────────────────────────────────
  合计: 14 项, 总大小 65.07 MB
```

## 功能特点

- 递归计算每个子目录的总大小
- 按大小**降序排列**，一眼看出最占空间的文件/目录
- 人类可读格式（B / KB / MB / GB / TB）
- 区分文件 📄 和目录 📁
- 最后汇总条目数和总大小
