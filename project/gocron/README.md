# gocron — 定时任务调度器

类 cron 的定时任务调度器，从 YAML 配置读取任务，支持并发执行、超时控制、执行日志，提供 HTTP 管理接口和 Web 面板。

## 快速启动

```bash
go run main.go -config tasks.yaml
```

浏览器打开 `http://localhost:8089` 查看管理面板。

## 参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `-config` | `tasks.yaml` | 任务配置文件 |
| `-port` | `8089` | HTTP 管理接口端口 |

## 配置文件格式

```yaml
tasks:
  - name: "打印时间"
    cron: "*/5 * * * * *"       # 秒 分 时 日 月 周（6字段）
    command: "date '+%H:%M:%S'"
    timeout: 10s

  - name: "磁盘检查"
    cron: "0 */1 * * * *"       # 每分钟
    command: "df -h / | tail -1"
    timeout: 30s
```

### Cron 表达式（6 字段）

```
┌─── 秒 (0-59)
│ ┌─── 分 (0-59)
│ │ ┌─── 时 (0-23)
│ │ │ ┌─── 日 (1-31)
│ │ │ │ ┌─── 月 (1-12)
│ │ │ │ │ ┌─── 周 (0-6, 0=周日)
│ │ │ │ │ │
* * * * * *
```

支持：`*`、`*/n`、`n-m`、`n,m,k`

## HTTP 接口

| 路径 | 说明 |
|------|------|
| `GET /` | Web 管理面板（自动刷新） |
| `GET /api/tasks` | 所有任务状态 JSON |
| `GET /api/history?name=xxx&limit=20` | 指定任务的执行历史 |

## 项目结构

```
gocron/
├── main.go                # CLI 入口
├── tasks.yaml             # 示例配置
├── config/config.go       # YAML 配置加载
├── cron/cron.go           # 6 字段 cron 表达式解析器
├── scheduler/scheduler.go # 核心调度器（并发+超时+日志）
└── api/api.go             # HTTP 接口 + Web 面板
```

## 功能特点

- **YAML 配置**：声明式任务定义，热重启即可生效
- **6 字段 cron**：支持秒级精度调度
- **并发执行**：每个任务独立 goroutine，互不阻塞
- **超时控制**：`context.WithTimeout`，超时自动 kill
- **执行日志**：终端彩色输出 + 内存保留最近 100 条记录
- **HTTP API**：JSON 接口查询任务状态和历史
- **Web 面板**：内嵌 HTML 页面，2 秒自动刷新
- **优雅停止**：Ctrl+C 信号处理
