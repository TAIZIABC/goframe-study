# serve — 静态文件服务器

指定目录和端口，启动一个 HTTP 静态文件服务器，支持目录浏览和彩色请求日志。

## 使用方式

```bash
go run main.go                          # 当前目录, 端口 8080
go run main.go -dir ./dist -port 3000   # 指定目录和端口
go run main.go /path/to/dir             # 第一个参数直接传目录
```

## 参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `-dir` | `.` | 要服务的目录路径 |
| `-port` | `8080` | 监听端口号 |
| `-host` | `0.0.0.0` | 监听地址 |

## 功能特点

- 内置**目录浏览**（Go 标准库 `http.FileServer`）
- 启动时打印 **localhost** 和**局域网 IP**，方便手机访问
- 彩色请求日志：状态码（绿/黄/红）、HTTP 方法、路径、耗时
- 读写超时保护（15s）
